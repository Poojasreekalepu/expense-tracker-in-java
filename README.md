# expense-tracker-in-java
Expense Tracker project in Java using OOP concepts and file handling

## Description

This is a simple Java-based Expense Tracker application. It helps users manage their daily expenses efficiently.

## Features

* Add expenses (amount, category, date, description)
* View all expenses
* Calculate total spending
* Filter expenses by category
* Data stored using file handling

## Technologies Used

* Java
* OOP (Object-Oriented Programming)
* ArrayList
* File Handling

## How to Run

1. Compile the program:
   javac expensetracker.java

2. Run the program:
   java expensetracker

## Sample Use

* Enter expense details through menu
* View and analyze spending
